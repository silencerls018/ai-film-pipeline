# 影调 / 人物打光知识来源（工程化摘要）

本目录为 **公开影视教学与工业通用术语** 的压缩，供 Agent 与提示词编译使用，**不是**付费课程全文。

## 公开参考

- StudioBinder — Three-point lighting；cinematic lighting techniques；motivated light language
- No Film School — Film lighting techniques primers（key/fill/back, high-key/low-key）
- Film-school face patterns — Rembrandt / loop / butterfly / split / broad / short
- Industry practice — practicals as key；protect eyes；contrast as emotion

## 工程文件

| 文件 | 内容 |
|------|------|
| `character_lighting.json` | 人物脸型光、情绪肖像配方、提示词优先级 |
| `lighting_for_emotion.json` | 七情绪 → 可见光效与氛围句 |
| `emotion_to_look.json` | 情绪 → 影调倾向 |
| `tone_types.json` | 高调/中调/低调 |

## 原则

教学概念 → 规则表；提示词只写**看得见的光**（方向、软硬、冷暖、半脸阴影），不写岗位黑话。
