# 07 · 资产 / 三视图（Asset）· 人维护

> 规范：`E:\AI\skill\三视图.skill` + 项目约定  
> AI：`ai/asset/three_view_template.json` · 代码：`runtime/three_view.py`

## 人物图怎么构图

```
┌──────────── 50% ────────────┬──────────── 50% ────────────┐
│  超大面部特写（几乎占满左半）  │  正面 / 背面 / 侧面全身      │
│  真皮肤、看镜头、写明「人种」  │  脸必须黑色方块遮挡          │
└─────────────────────────────┴─────────────────────────────┘
                         中性灰背景
```

## 必须包含

1. **人种 / Ethnicity**（如 East Asian）  
2. 左半超大脸  
3. 右半三身位 + 黑块遮脸  

## 主稿语言

- 生图：**全英文 `sheet_prompt`**  
- 给你看：中文 `sheet_prompt_zh_summary`  

## 剧本没写外貌时

按风格包 + 声口推断发型服装，中文里标注「可改」；人种默认可按东亚，也可在角色卡写 `ethnicity`。
