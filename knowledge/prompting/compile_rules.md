# Prompt compile rules

1. **Assemble, don't invent** — only use fields present in FilmBible shot contracts.
2. Prefer structured facts (40mm, slow_push_in, low_key) over vague adjectives alone.
3. Always keep movement.motivation and look.motivation in the prompt or summary.
4. Put scene/style `forbidden` items into negative_prompt.
5. Dialogue becomes performance cues, not burned-in subtitles.
6. Export both machine prompts and zh_director_summary for human review.
