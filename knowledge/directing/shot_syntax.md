# Shot syntax

Common narrative patterns:

1. Establish (WS/EWS) → relationship (MS) → detail (CU/INSERT) → reaction (MCU/CU)
2. Dialogue coverage: master → OTS A → OTS B → clean singles for pressure
3. Revelation: hold on face, then insert of object, or reverse (object then face)
4. Avoid talking-heads only; interleave inserts and reactions

Shot size codes: EWS, WS, FS, MS, MCU, CU, ECU, INSERT
